183  Run one timestep of the environment's dynamics. When the end of the

186

188  @returns

209  self._spine_spine.set_action(spine_action)
210 
211  # Observe

213  observation = self.get_env_observationget_env_observation(spine_observation)
214  reward = self.get_rewardget_reward(observation, action)
