30  Environment where Upkie is used as a wheeled inverted pendulum.
31 
32  The environment id is ``UpkieGroundVelocity-v3``. Model assumptions are
33  summarized in the <a
34  href="https://scaron.info/robotics/wheeled-inverted-pendulum-model.html">following
35  note</a>.
36 
37  @note For reinforcement learning with neural networks: the observation
38  space and action space are not normalized.
39 
40  ### Action space
41 
42  The action corresponds to the ground velocity resulting from wheel
43  velocities. The action vector is simply:
44 
45  <table>
46  <tr>
47  <td><strong>Index</strong></td>
48  <td><strong>Description</strong></td>
49  </tr>
50  <tr>
51  <td>``0``</td>
52  <td>Ground velocity in [m] / [s].</td>
53  </tr>
54  </table>
55 
56  Note that, while this action is not normalized, [-1, 1] m/s is a reasonable
57  range for ground velocities.
58 
59  ### Observation space
60 
61  Vectorized observations have the following structure:
62 
63  <table>
64  <tr>
65  <td><strong>Index</strong></td>
66  <td><strong>Description</strong></td>
67  </tr>
68  <tr>
69  <td>0</td>
70  <td>Pitch angle of the base with respect to the world vertical, in
71  radians. This angle is positive when the robot leans forward.</td>
72  </tr>
73  <tr>
74  <td>1</td>
75  <td>Position of the average wheel contact point, in meters.</td>
76  </tr>
77  <tr>
78  <td>2</td>
79  <td>Body angular velocity of the base frame along its lateral axis,
80  in radians per seconds.</td>
81  </tr>
82  <tr>
83  <td>3</td>
84  <td>Velocity of the average wheel contact point, in meters per
85  seconds.</td>
86  </tr>
87  </table>

92 
93  - ``leg_return_period``: Time constant for the legs (hips and knees) to

300  def get_reward(
301  self,
302  observation: NDArray[float],
303  action: NDArray[float],
304  ) -> float:
305  """!
306  Get reward from observation and action.
307 
308  @param observation Environment observation vector.
309  @param action Environment action vector.
310  @returns Reward.
311  """
312  pitch = observation[0]
313  ground_position = observation[1]
314  angular_velocity = observation[2]
315  ground_velocity = observation[3]
316 
317  tip_height = 0.58 # [m]
318  tip_position = ground_position + tip_height * np.sin(pitch)
319  tip_velocity = (
320  ground_velocity + tip_height * angular_velocity * np.cos(pitch)
321  )
322 
323  std_position = 0.05 # [m]
324  position_reward = np.exp(-((tip_position / std_position) ** 2))
325  velocity_penalty = -abs(tip_velocity)
326 
327  return (
328  self.reward_weightsreward_weights.position * position_reward
329  + self.reward_weightsreward_weights.velocity * velocity_penalty
