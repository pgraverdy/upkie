Detailed Description

Environment where Upkie is used as a wheeled inverted pendulum.

The environment id is UpkieGroundVelocity-v3. Model assumptions are summarized in the following note.

Note
For reinforcement learning with neural networks: the observation space and action space are not normalized.

Action space

The action corresponds to the ground velocity resulting from wheel velocities. The action vector is simply:

Index Description
0 Ground velocity in [m] / [s].

Note that, while this action is not normalized, [-1, 1] m/s is a reasonable range for ground velocities.

Observation space

Vectorized observations have the following structure:

Index Description
0 Pitch angle of the base with respect to the world vertical, in radians. This angle is positive when the robot leans forward.
1 Position of the average wheel contact point, in meters.
2 Body angular velocity of the base frame along its lateral axis, in radians per seconds.
3 Velocity of the average wheel contact point, in meters per seconds.

Attributes

The environment class defines the following attributes:

  • leg_return_period: Time constant for the legs (hips and knees) to revert to their neutral configuration.
  • version: Environment version number.
  • wheel_radius: Wheel radius in [m].
